# To-Do List Web Application

This is a simple to-do list web application built using HTML, CSS, and JavaScript. It allows users to add tasks, mark them as completed, set due dates, and delete tasks.

## Features

- Add new tasks with due dates.
- Mark tasks as completed by checking them off.
- Delete tasks individually.
- Simple and intuitive user interface.
- Support for HTML attribute validation and DOM event-based validation.

## Usage

1. Clone the repository:

```
git clone <repository_url>
```

2. Open the `index.html` file in your web browser.

3. Enter a task description in the input field and optionally set a due date.

4. Press the "Add Task" button to add the task to the list.

5. Check off tasks by clicking the checkbox next to them.

6. Delete tasks by clicking the "Delete" button.
